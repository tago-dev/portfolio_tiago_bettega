## 🚀 Como Executar

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/tago-dev/portfolio.git
   cd portfolio
   ```

2. **Abra o arquivo**:
   - Abra `index.html` diretamente no navegador, ou
   - Use um servidor local como Live Server (VS Code)

3. **Para desenvolvimento**:
   ```bash
   # Com Python
   python -m http.server 8000
   
   # Com Node.js
   npx serve .
   
   # Com PHP
   php -S localhost:8000
   ```

## 🎨 Personalização

### Cores
As cores podem ser facilmente alteradas modificando as CSS Custom Properties no arquivo `style.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #06b6d4;
    --accent-color: #f59e0b;
    /* ... outras variáveis */
}
```

### Conteúdo
- Edite as informações pessoais no `index.html`
- Substitua a foto de perfil em `assets/foto.jpg`
- Atualize os links das redes sociais
- Modifique os projetos e experiências

### Funcionalidades
- Adicione novas seções seguindo a estrutura existente
- Implemente novas animações no `script.js`
- Customize os efeitos visuais no CSS
